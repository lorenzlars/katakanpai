## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm (recommended)

### Installation

```bash
# Install dependencies
pnpm install
```

### Development

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

### Production

Build the application for production:

```bash
pnpm build
```

Preview the production build:

```bash
pnpm preview
```

## How to Play

1. Click "Start" to begin the game
2. A katakana brand name will appear
3. Try to guess the brand before the timer runs out
4. Click "Reveal" to see the answer
